RELATIVE
usando questa proprietà tutti i discendenti (figli e nipoti ) che usano un valore postion diverso da relative si riferiranno al contenitore in oggetto (le coordinate sono vincolate a questo componetne),
ABSOLUTE
La proprietà position absolute nei CSS consente di rimuovere un elemento dal flusso naturale della pagina, seguendo quanto specificato nelle proprietà top, bottom, z-index, left e right.
FIXED
usando questa prprietà posiziono l'elemento dove voglio e questo non è sensibilie allo scorrimento della pagina.
STICKY
usando questa proprietà ogni volta che scorro la pagina l'elemento si alza fino a quando non tocca il bordo superiore del padere
